Change Management Portal developed in MERN Stack for internal use by Cyber Security Operations Centre (CSOC), AAI.

Usage Instructions:
1. Start the database by running **brew services start mongodb/brew/mongodb-community** (Mac) or **sudo systemctl start mongodb** (Linux) or **net start MongoDB** (Powershell)
2. Run **node server.js**
3. cd client/
4. Run **npm run dev**
